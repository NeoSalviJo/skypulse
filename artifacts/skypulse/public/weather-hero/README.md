## Weather hero matte plates

Layered PNG artwork used by the fullscreen **`LivingSkyAtlas`** (ambient backdrop) and kept for alternate hero compositions (`cinematic-hero-scene.tsx` when enabled).

- `wx-plate-clear-day.png` — primary daytime sky atlas
- `wx-plate-clear-night.png` — night atmosphere
- `wx-plate-storm.png` — storm / thunderhead mood
- `wx-layer-cloud-mid.png` — mid-depth volumetric cloud mass (screen/blend layered)
- `wx-layer-cloud-fg.png` — foreground framing clouds
- `wx-layer-godrays.png` — warm volumetric shafts (additive / screen)
- `wx-moon.png` — lit moon disk for clear night

Assets were produced as illustrative reference art for Skypulse. Replace paths in `weather-hero-assets.ts` if you swap imagery.
